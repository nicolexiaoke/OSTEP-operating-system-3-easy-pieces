##👉Framework运行方法

1. 下载并解压Framework.zip至文件夹Framework

2. 打开window命令行，并在命令行小黑框中进入文件夹Framework

3. 分别输入一下三句：

  set FLASK_APP=flaskr
  
  set FLASK_ENV=development
  
  flask run
  
4. 打开http://127.0.0.1:5000/auth/register 或 http://127.0.0.1:5000/auth/login
